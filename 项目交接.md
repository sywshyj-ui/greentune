## 核心功能
- 本地音乐:导入文件/文件夹、拖拽添加、ID3 元数据(GBK 乱码修复)、封面
- 歌词:本地 .lrc(UTF-8/GBK 自动解码)+ 在线歌词,逐行同步高亮
- 在线音乐:QQ 音乐搜索/试听/歌词 + 网易云歌单导入(NetEase 元数据 + QQ 音源)
- 播放控制、播放列表、搜索(Ctrl+K)、多选删除、右键菜单
- 系统托盘:绿色音符图标,双击显示窗口

## 关键技术坑(避免重踩)
- **HTTP 中文乱码**:必须 `Buffer.concat(chunks).toString('utf8')` 在 `end` 时整体解码,不能逐块解码(多字节中文会被切断)
- **网易云歌单截断**:完整 ID 在 `trackIds`,不是 `tracks`
- **Electron net 下载被拦**:`ERR_BLOCKED_BY_CLIENT` → 改 Node 原生 https/http
- **Google 翻译国内不通** → 有道 demo 接口
- **跨域音频污染 Web Audio** → 在线播放无真实频谱,只能假跳动
- localStorage 用 `LS` 助手,键前缀 `gt_`;在线歌单存 `gt_onlinePlaylists`
